# dexamethasone suppression test

- give 1 mg at 11 p.m.;
- ✓ 8 a.m. serum cortisol (suppression if <1.8 µg/dL);
- <5% false ⊕ (primarily used to evaluate subclinical Cushing’s in adrenal “incidentalomas”)
