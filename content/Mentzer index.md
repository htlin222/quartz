# Mentzer index

> mean corpuscular volume/red blood cell : 以13為界

- 分子 = mean corpuscular volume = 質 = 質差= 地中海 = 分子小= index 小
- 分母 = red blood cell = 量 = 量少= 缺鐵 = 分母小=  index 大 = 比13大
