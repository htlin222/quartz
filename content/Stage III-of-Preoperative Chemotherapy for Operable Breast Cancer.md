# Stage III-of-Preoperative Chemotherapy for Operable Breast Cancer

## HR(+) HER2(-)

- AC X 4 →  Taxotere X 4 → surgery
  - ACT手術 c.f. II: AC手術T
- if PCR:
  - **no Adjuvant**
- else:
  - [[Capecitabine]] x 8

## TNBC

> 基本上 Stage II and Stage III 做法一樣
> [[Triple negative]]

- AC + Pembrolizumab x 4 →Taxotere/CDDP+ Pembrolizumab x 4→surgery →
- if PCR:
  - [[Pembrolizumab]] x 9
- else:
  - [[Capecitabine]] x 8
