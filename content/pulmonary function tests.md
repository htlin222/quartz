- RV: residual volume,
- ERV: Exspiratory reserve volume,
- IRV: Inspiratory reserve volume
- FRC (Functional residual capacity) = ERV + RV
- FVC (Functional vital capacity) = IRV + TV+ ERV TLC (Total lung capacity) = IRV+ TV+ ERV +RV
