# SVC syndrome

(NEJM 2007;356:1862): central tumor → SVC compression → face/arm swelling (>80%), neck/chest vein distention (~60%), SOB/cough (~50%), HA (~10%); Rx = steroids, diuretics, RT ± chemo, SVC stent if severe sx, anticoag if clot
