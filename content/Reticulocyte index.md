# Reticulocyte index

缺血時 R 佔比天然地變高， ∴ 因此要給他打折才 ✖ 不會誤以為骨髓生產力好

### RI =

[reticulocyte count × (Pt’s Hct/nl Hct)]/maturation factor]

### maturation factors for a given  (天) 

- Hct: 45% = 1,
- 35% = 1.5,
- 25% = 2,
- 20% = 2.5

> RI >2% → adequate marrow response; RI <2% → hypoproliferation
