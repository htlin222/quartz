# Coxiella burnetii

- Q熱是貝氏考克斯菌（Coxiella burnetii）
- 所引起之人畜共通傳染病（zoonosis），
- 可分為急性Q熱及慢性Q熱。
- 動物感染Q熱通常沒有症狀，
- 但可能會造成懷孕的動物流產、
- 死產或不孕；
- 這些動物的胎盤組織、羊水、死胎及子宮排出物等含有大量的病原體，若未妥善處理且隨意掩埋，
即可能污染環境而成為人類感染Q熱的重要感染源。
- 獸醫、毛皮業者、農場員工、屠宰場員工、畜牧業者及相關研究人員等，尤其是經常接觸牲畜（山羊、綿羊、牛隻等）的工作者，
- 為Q熱的高危險群。
